About Annonatate

Annonatate is an application that allows you to create W3 web annotations on images. W3 Web Annontations is an open source standard for creating annotations. Because of it's open source nature these annotations can be used with many other tools. Annonatate also is compatible with IIIF images and/or IIIF manifests.
